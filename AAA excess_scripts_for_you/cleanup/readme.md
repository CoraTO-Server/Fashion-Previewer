# Cleanup Scripts

Scripts for managing BMP/PNG files and folder structures.

## Scripts

### cleanup_bmp_files.py
- **Purpose**: Clean up BMP files after conversion
- **Dependencies**: None (uses built-in modules only)
- **OS Support**: Windows/Linux (has .bat and .sh launchers)

### delete_bmp_rename_folder.py
- **Purpose**: Rename "BMP" folders to "PNG" and remove BMP files
- **Dependencies**: None (uses built-in modules only)
- **Usage**: `python delete_bmp_rename_folder.py [directory]`

## Dependencies
These scripts use only built-in Python modules:
- os
- pathlib
- shutil
- sys

No external dependencies required - safe to run without installation!
