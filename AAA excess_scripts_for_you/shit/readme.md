# Palette Management Scripts

Scripts for managing and reordering palette files. This thing is evil, and caused many bugs
    for the icon editor. I do not suggest it. However, you may need it, for something else,
    for all I know, so enjoy.

## Scripts

### reorder_palettes_light_to_dark.py
- **Purpose**: Sort RIFF palette colors from brightest to darkest
- **Dependencies**: None (uses built-in modules only)
- **Built-in modules used**: struct, pathlib, colorsys, argparse
- **Usage**: `python reorder_palettes_light_to_dark.py [input.pal]`

## Dependencies
These scripts use only built-in Python modules:
- struct
- pathlib
- colorsys
- argparse

No external dependencies required - safe to run without installation!
